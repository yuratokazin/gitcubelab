## Создание репозитория на GitHub

### Шаг 1. Регистрация на GitHub

1. **Создайте аккаунт** на GitHub:
   - Укажите логин, email и пароль
   - Подтвердите email
   - Заполните профиль (опыт, навыки, образование)

### Шаг 2. Создание репозитория

1. **Перейдите** на главную страницу GitHub
2. Нажмите на значок **+** в правом верхнем углу
3. Выберите **New repository**

### Шаг 3. Настройка параметров репозитория

1. **Основные параметры**:
   - **Имя репозитория** — уникальное название
   - **Описание** — краткое описание проекта
   - **Тип доступа**:
     - Public (публичный) — доступен всем
     - Private (приватный) — только для выбранных пользователей
2. **Дополнительные опции**:
   - **README.md** — создайте файл с описанием проекта
   - **.gitignore** — выберите шаблон для игнорирования файлов
   - **Лицензия** — выберите тип лицензии для проекта

### Шаг 4. Загрузка проекта

Есть несколько способов загрузить проект:

1. **Через GitHub Desktop**:

   - Установите приложение
   - Авторизуйтесь
   - Выберите **Clone repository**
   - Укажите путь для сохранения

2. **Через терминал**:

   - Перейдите в папку с проектом

   - Выполните команды:

     bash

1. - ```bash
     git init
     git add .
     git commit -m "Initial commit"
     git remote add origin <ссылка_на_репозиторий>
     git push -u origin main
     ```

2. **Через VS Code**:

   - Авторизуйтесь в GitHub через VS Code
   - Откройте Source Control
   - Нажмите Initialize Repository
   - Выберите Publish to GitHub



После создания репозитория вы можете начать добавлять файлы и работать над  проектом как через веб-интерфейс GitHub, так и локально на компьютере.
