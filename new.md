## Настройка режима «Картинка в картинке» на Debian с одним монитором

### Что такое PiP на одном мониторе

**PiP (Picture-in-Picture)** — это технология, позволяющая отображать два окна на одном экране, где  одно окно отображается в уменьшенном виде поверх основного.

### Необходимые инструменты

- **Xrandr** — утилита для управления дисплеями
- **Arandr** — графический интерфейс для xrandr
- Оконный менеджер (GNOME, KDE или другой)

### Пошаговая настройка

1. **Проверка текущего состояния**:

bash

```bash
xrandr
```

1. **Установка Arandr** (для графического управления):

bash

```bash
sudo apt-get install arandr
```

1. **Создание виртуального окна**:

bash

```bash
xrandr --output eDP-1 --mode 1920x1080 --fb 3840x2160
```

### Настройка через терминал

1. **Создание нового режима**:

bash

```bash
xrandr --newmode "1920x1080_60.00" 173.00  1920 2048 2248 2576  1080 1083 1088 1120 -hsync +vsync
```

1. **Добавление режима**:

bash

```bash
xrandr --addmode eDP-1 1920x1080_60.00
```

1. **Применение настроек**:

bash

```bash
xrandr --output eDP-1 --mode 1920x1080_60.00
```

### Управление окнами

1. **Запуск приложений в разных окнах**:

bash

```bash
DISPLAY=:0.0 приложение1 &
DISPLAY=:0.1 приложение2 &
```

1. **Изменение размера окон**:

bash

```bash
wmctrl -r "название_окна" -e 0,X,Y,W,H
```

### Графическая настройка через Arandr

1. Запустите **Arandr**:

bash

```bash
arandr
```

1. Настройте расположение окон в графическом интерфейсе
2. Сохраните конфигурацию

### Автоматизация

Создайте скрипт для автоматического запуска:

bash

```bash
#!/bin/bash
xrandr --output eDP-1 --mode 1920x1080 --fb 3840x2160
wmctrl -r "окно1" -e 0,0,0,1920,1080
wmctrl -r "окно2" -e 0,1920,0,1920,1080
```



### Рекомендации

- Используйте **tmux** или **screen** для управления сессиями
- Настройте автозапуск через:

bash

```bash
sudo nano ~/.config/autostart/pip-setup.sh
```

- Создайте резервную копию текущих настроек перед изменениями:

bash

```bash
xrandr > xrandr.backup
```
