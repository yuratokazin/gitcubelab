---



Чтобы бургер‑меню стало **видимым** и **работало**, нужны три компонента:

1. **HTML** — правильная разметка с чекбоксом и меткой;
2. **CSS** — стили для иконки и переключения меню;
3. **JavaScript** — логика открытия/закрытия меню при клике.

Ниже — полный рабочий код.

------

### 1. Исправленный HTML (в шапке)

Замените вашу кнопку на эту структуру:

```html
<header class="header">
  <div class="container">
    <a href="#" class="logo">Logo</a>


    <!-- Чекбокс для управления меню -->
    <input type="checkbox" id="menu-toggle" class="menu-toggle">
    <!-- Метка-иконка бургера -->
    <label for="menu-toggle" class="burger-menu">☰</label>


    <!-- Навигационное меню -->
    <nav class="nav">
      <a href="#" class="nav-link">Главная</a>
      <a href="#" class="nav-link">Курсы</a>
      <a href="#" class="nav-link">О нас</a>
      <a href="#" class="nav-link">Контакты</a>
    </nav>
  </div>
</header>
```

------

### 2. Дополненный CSS

Добавьте эти стили в ваш CSS (после существующих):

```css
/* Скрываем чекбокс */
.menu-toggle {
  display: none;
}

/* Стили бургер-меню */
.burger-menu {
  display: none;
  background: none;
  border: none;
  font-size: 1.5rem;
  cursor: pointer;
  color: #333;
  padding: 0.5rem;
}

/* Показываем бургер на мобильных */

    display: block;
    position: absolute;
    right: 15px;
    top: 15px;
  }

  .nav {
    display: flex;

  }

/* Состояние: меню открыто */
.menu-toggle:checked + .burger-menu {
  color: #2563eb;
}

.menu-toggle:checked ~ .nav {
  transform: translateX(0);
}
```

------

### 3. JavaScript (обязательно!)

Добавьте этот скрипт в конец HTML‑файла (перед `</body>`):

```html
<script>
  // Находим элементы
  const menuToggle = document.getElementById('menu-toggle');
  const burgerMenu = document.querySelector('.burger-menu');

  // При клике на бургер — переключаем чекбокс
  burgerMenu.addEventListener('click', () => {
    menuToggle.checked = !menuToggle.checked;
  });

  // Закрываем меню при клике на ссылку (опционально)
  document.querySelectorAll('.nav-link').forEach(link => {
    link.addEventListener('click', () => {
      menuToggle.checked = false;
    });
  });
</script>
```

------

### Что это даёт

1. **Видимость**
   - Бургер показывается только на экранах ≤ 768 px.
   - Иконка (`☰`) контрастная, кликабельная.
2. **Работа**
   - При клике на бургер меню выезжает справа.
   - При повторном клике — скрывается.
   - Клики по ссылкам тоже закрывают меню.
3. **Плавность**
   - Анимация выезда/уезда (0.3 с).
   - Тень и фон для лучшего восприятия.
4. **Доступность**
   - Чекбокс скрыт, но управляется через `<label>`.
   - Экранные читалки видят элементы.

------

### Проверка

1. Откройте страницу на мобильном или в режиме адаптива (F12 → мобильный режим).
2. Нажмите на ☰ — меню должно выехать справа.
3. Нажмите снова — меню уберётся.
4. Кликните по ссылке — меню закроется.

Если что‑то не работает, проверьте:

- нет ли конфликтов с другими стилями;
- подключён ли JavaScript;
- совпадают ли классы в HTML/CSS/JS.
